# 用户音乐品味

> DJ 视角的 Tree 画像，不是数据分析报告。

## 我了解的 Tree

Tree 有两套完全不同的听歌逻辑：

1. **学习/阅读时** —— 开纯音乐当背景音。钢琴、新古典、日系配乐，节奏稳定不抢注意力。这部分是收藏别人的歌单，**主要用途是"不听得无聊但又不能分心"**，这是工具，不是真爱。

2. **自己主动听** —— Like + 念欢 是他真正花时间听的歌单。这两才是他个人品味的体现：
   - **Like**：欧美流行为主，Justin Bieber、Taylor Swift、Ariana Grande、Post Malone 都是常客
   - **念欢**：中文抒情+独立音乐为主，G.E.M.邓紫棋、告五人、h3R3、蔡健雅为核心，偶尔来首小众说唱（队长、Cubi）

平时不追榜单，不跟风，有自己的体系。

## 推荐艺术家

**Like 歌单核心：**
- Justin Bieber、Taylor Swift、Ariana Grande、Post Malone
- Ed Sheeran、Billie Eilish、Mariah Carey
- AURORA（北欧空灵嗓，Like 和 念欢 都有她）
- In Love With A Ghost（氛围纯音，Like 里有）

**念欢歌单核心：**
- G.E.M.邓紫棋、告五人、h3R3、蔡健雅
- 颜人中、陈粒、朴树、单依纯
- 汪苏泷、队长、海洋Bo
- 很美味（独立清新）

**学习背景音乐（工具型）：**
- Alexis Ffrench、西村由紀江、坂本龍一
- 広橋真紀子、久石譲、Kevin Kern

## 隐含规则（容易踩的坑）

- Tree 说"随便放一首"意思是 **1-2首**，不是歌单
- Tree 在学习/刷题状态时，**稳定压过一切**，不要突然来首气氛太跳跃的
- Tree 说"来点刺激的"——问他是指学习时的专注型，还是想换换心情，两种完全不同的方向
- **Like + 念欢 的歌优先推荐**，这两的歌单才是他真正认可的品味
- 纯音乐不等于 Tree 的品味——是工具，不是真爱

## 场景直觉

- 学习/刷题 → 纯音乐当背景音（功能性）
- 自己主动想听 → **Like + 念欢** 里选（欧美流行/中文抒情）
- 放松/放空 → Lo-Fi、Ambient
- 夜间思考 → 极简钢琴、氛围
- 早晨开场 → 轻柔钢琴或轻流行

## 不感兴趣

摇滚、主流电子舞曲、抖音神曲批量跟风。
如果 Tree 主动提到想要"不一样"，可以小试探，但别突然大转向。
